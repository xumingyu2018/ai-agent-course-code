# 深度调研助手 — 用户偏好

本文件在 Agent 启动时作为长期记忆加载。

## 报告偏好

- **所有报告、笔记、任务列表均使用中文**
- 报告顶部包含「执行摘要」（3–5 条要点）
- Markdown 标题层级：`#` 标题、`##` 章节、`###` 小节
- 每份报告末尾必须有「参考资料」章节，列出所有引用 URL
- 正文以段落叙述为主；仅在摘要或对比表时使用 bullet 列表

## 调研标准

- 重要结论尽量用两个以上独立来源交叉验证
- 明确标注信息冲突，不要掩盖分歧
- 区分事实与推测/预测
- 时效性信息需标注日期

## 工作区目录

- 调研计划：`/workspace/sources/research_plan.md`
- 各子主题原始资料：`/workspace/sources/findings_*.md`
- 报告草稿：`/workspace/reports/draft_*.md`
- 最终报告：`/workspace/reports/report_*.md`
